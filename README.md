## My Contribution

This project was developed as part of a senior design team at Embry-Riddle Aeronautical University building a On-Off-Key Optical CubeSat Communications Payload using Commercial-off-the-Shelf (COTS) Components

My contributions included:
- Designed and implemented intially TCP but changed to UDP-based optical communication system in Python
- Performed throughput, latency, and packet loss testing
- Worked with SFP modules and fiber/free-space optical setup
- Debugged link issues and optimized performance (~800 Mbps achieved)
- Created data packet encasulation within Python to transmit data across fiber optic -> then to free space
