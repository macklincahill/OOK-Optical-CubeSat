## My Contribution

*This project was developed as part of a senior design team at Embry-Riddle Aeronautical University building a On-Off-Key Optical CubeSat Communications Payload using Commercial-off-the-Shelf (COTS) Components*

My contributions included:
• Designed and implemented a 1 Gbps free-space optical communication payload using On-Off Keying (OOK) with Commercial-Off-The-Shelf (COTS) components. 
• Achieved high-speed optical data transmission measured by 800 Mbps with less than 10% packet loss by creating a User Datagram Protocol (UDP) optical system in Python on NVIDIA Jetson Orin Nano using Ubuntu Linux. 
• Integrated fiber and optical free space hardware using both a fiber-based system and an optical-based system with the Intel I350 LRES2203PF-2SFP network cards with 1550 nm SFP transceivers and fiber collimators measured by data transmission across 8 cm of free-space whilst maintaining data rates and packet loss. 
• Created and debugged optical linkage, resolving link failures related to attenuation, and beam alignment while increasing system working distance. 
• Optimized network performance through MTU tuning, packet sizing, system timeout, packet loss flagging to improve throughput and reduce packet loss, while maintaining data integrity.
